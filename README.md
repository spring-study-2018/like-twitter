# Like Twitter

## Requirements
- JDK >= 8
- [Gradle](https://gradle.org/install/)
- Docker

## Quick Start

#### Build
``` sh
gradle clean build
```
#### Bake Docker Images
``` sh
source JenkinsFile.sh
```

#### Run
``` sh
docker-compose up
```
